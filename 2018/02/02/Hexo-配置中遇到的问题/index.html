Hexo 配置中遇到的问题

2018-02-02 Blog, Hexo, Themes 0 Comments

在配置 Hexo 过程中遇到的问题及解决方法

主要修改的有 Hexo 的 _config.yml 和 Themes 主题文件中的 _config.yml,用博客配置文件和主题配置文件代指这两个.yml

Themes

主题文件配置过程中，需要修改博客配置文件的 theme,注意 theme后的主题文件名需要小写，而且必须和 Themes 文件夹中的主题名一致

网页图片

我在用的几个Themes中，主题配置文件中的图片名是不能更改的。修改背景图片、头像等，要把原来的图片删除或重命名，再将新的图片导入主题目录，必须和被替代的图片目录、名称保持一致。
