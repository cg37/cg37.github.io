2. 自动化表单

在工作中发现传统的表单组建易用性差，造成了开发工作的重复和浪费，我用 React + BootStrap 写了几个可实现自动化部署的表单组件，减少前端代码的开发量。有：根据后台发送的数据自动生成表单；根据用户的输入自定义表单组件的类型和数量，在用户输入表单内容后自动更新数据单，用户点击提按钮就可以一键提交所有数据，并加入了数据验证防止提交是数据丢失。[部分代码地址](https://github.com/cg37/React-Component)
